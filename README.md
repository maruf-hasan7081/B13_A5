1.In JavaScript, var, let, and const are all used to declare variables, but their behavior is different.

var is the older method. It follows function scope and the same variable can be declared multiple times. It also supports var hoisting, meaning that it can be used even before it is declared.

let was added in ES6. It follows block scope, meaning that if declared inside {}, it cannot be used outside that block. The same variable cannot be declared again, but its value can be changed.

const was also added in ES6 and it also follows block scope. The value of a variable declared with const cannot be changed later. Const is usually used for variables whose value remains constant.

2.Spread operator (...) is a modern feature of JavaScript that helps to spread the elements of an array or object. Using it, you can easily copy, merge, or add new data to an array or object. This makes it easier to manipulate data with less code.

3.map(), filter(), and forEach() are JavaScript array methods that are used to operate on each element of an array.

map() creates a new array by performing an operation on each element.
filter() creates a new array by selecting some elements according to a specified condition.
forEach() loops over each element of an array, but it does not return a new array. It is usually used only for iteration or side effects.

4.Arrow functions are a shorthand syntax for JavaScript functions that was added in ES6. They allow you to write functions that are smaller and simpler than regular functions. Arrow functions make code cleaner and more readable, and they are widely used in modern JavaScript.

5.Template literals are a modern way of writing strings in JavaScript that uses backtick ( ). This allows you to easily use variables or expressions inside a string. Template literals also make it easier to write multiline strings and make string formatting clearer.
